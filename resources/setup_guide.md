# Setup Guide

## Install Anaconda (Windows/Mac/Linux)
1. Go to [anaconda.com/download](https://www.anaconda.com/download)
2. Install Python 3.11 or higher
3. Launch Anaconda Navigator → Jupyter Notebook

## Install Tableau Public
1. [Download here](https://public.tableau.com/app/download)
2. Free account required
3. No license needed

## Verify installation
Open terminal/command prompt:
`conda --version`
`jupyter --version`
`python --version`

## Clone this repo
`git clone https://github.com/your-username/data-analytics-pilot.git`
