# Data Analytics Pilot Program

**Duration:** 4 weeks  
**Instructor:** Amos Augo  
**Prerequisites:** Basic computer literacy

## How this repo works
- Each week has a folder with slides, exercises, and data.
- Do exercises in order. Solutions are in `/solutions/` folders.
- For questions: [Open an Issue](link-to-issues)
- Final project submission: Fork this repo, complete `week4-capstone/submission_template/`, then open a Pull Request.
